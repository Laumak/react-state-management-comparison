# @repo/ui

Shared React UI components used by all apps in this monorepo.

The goal of this package is to ensure every app renders exactly the same interface so that
comparisons focus on the state management integration, not on UI differences.

## What belongs here

- `TodoItem` — single to-do row (checkbox, label, delete button)
- `TodoList` — list container + empty state
- `TodoInput` — controlled input + "Add" button
- `PokemonCard` — displays a Pokémon's name, sprite, and types
- `PokemonList` — grid/list wrapper for Pokémon cards
- Layout primitives (Page, Section, etc.)
